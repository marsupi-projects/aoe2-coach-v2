## Decisions made

| Area | Decision | Reasoning |
|---|---|---|
| `knowledge/` location | Moved to project root (Option B) — out of `data/` entirely | `data/` is runtime output; civ JSON files are human-authored source material and belong in version control like source code |
| Docker client selection | Conditional in `database.py`: `HttpClient` when `CHROMA_HOST` is set, `PersistentClient` otherwise | Local dev stays unchanged; Docker mode is activated by setting `CHROMA_HOST=chromadb` in `.env` |
| Volume strategy | Bind mounts: `./data:/app/data` for agent, `./data/chroma:/chroma/chroma` for ChromaDB | User can inspect and back up all data directly from the host filesystem |
